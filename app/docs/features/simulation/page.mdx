# Simulation Mode

Simulation mode is one of the most important parts of CryptoDesk AI. It gives users a realistic
workflow without exposing real funds.

<h2 id="virtual-balance">Virtual Balance</h2>

The alpha starts with a **$10,000 virtual balance** so users can:

- practice order entry
- understand position sizing
- test the interface safely
- compare AI guidance against their own decisions

<h2 id="paper-trading-flow">Paper Trading Flow</h2>

Simulation mode mirrors the rhythm of trading without moving actual exchange balances. Users can
create test orders, review estimated notional impact, and build confidence in the workflow.

<h2 id="history-and-reset">History and Reset</h2>

The simulation workspace is meant to be explored repeatedly, so it includes:

- simulation trade history
- win-rate and P&L context
- reset behavior for a clean restart
- fast refresh actions for local state review

<Callout tone="success" title="Recommended first path">
If someone is new to CryptoDesk AI, simulation mode should be the first serious workflow they use.
It is the safest way to understand the product and give meaningful feedback.
</Callout>
